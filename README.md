# osu!dl

Search, filter, and bulk download osu beatmaps!

![osu!dl Promo](/apps/web/public/assets/images/promo.png)

Also check out [osu!complete](https://osucomplete.org)! A lot of its source code is being reused here.

## Features

- Full-text search within all attributes of beatmaps
- Advanced search filters
- Stores all ranked, approved, and loved maps, including those rendered undownlaodable by osu! due to copyright
    - Copyrighted maps are stored in a playable state but without audio and image assets
- Extremely fast bulk map downloads directly to a device folder
- Fallback streamed zip bulk downloads
- Modern and responsive web UI ideal for use on both desktop and mobile
- Optional desktop app providing the full feature set without the need for a browser
- Off-site S3-compatible storage model
